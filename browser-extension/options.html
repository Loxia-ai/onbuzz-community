Where is my token?

On first run of OnBuzz, a token is generated and stored at <OnBuzz user data dir>/settings/extension-token.json. Open that file and copy the token field. You can also set the ONBUZZ_EXTENSION_TOKEN env var on the server — that takes precedence over the file.

Full instructions: docs/BROWSER_EXTENSION.md in the OnBuzz repo.
