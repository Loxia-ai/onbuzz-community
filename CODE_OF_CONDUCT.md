# Code of Conduct

OnBuzz Community follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: **be kind, be patient, be honest about what you don't know, and assume the other person is acting in good faith** until clear evidence proves otherwise.

We expect this in:

- GitHub issues and pull requests
- Discussions
- Code review feedback
- Any other space where people identify themselves as part of this community

If you experience or witness behavior that violates this — harassment, discrimination, sustained personal attacks, or anything that makes contributors feel unwelcome — please email **`contact@loxia.ai`**. Reports are read by a small group of maintainers and treated confidentially.

We will, in order of preference: ask the person to stop, give them a temporary ban from project spaces, or remove them entirely.

This applies to maintainers too. If a maintainer is the problem, email the same address — the report won't be routed to them.
